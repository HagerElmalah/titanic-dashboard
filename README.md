# 🚢 Titanic Survival Analysis Dashboard

## 📖 Overview
This project explores the Titanic dataset to analyze survival patterns among passengers.  
The objective is to identify key factors affecting survival rates such as gender, age, and passenger class, and present insights through a clear and interactive dashboard.

---

## 🛠 Tools & Technologies
- Microsoft Excel  
- Pivot Tables  
- Data Visualization (Charts)  
- Conditional Formatting  

---

## 📊 Key Insights
- Female passengers had a higher survival rate compared to males  
- First-class passengers were nearly 3 times more likely to survive than third-class passengers  
- Survival rates were influenced by both age and passenger class  

---

## 📁 Dataset
- Titanic dataset (~1000 passenger records)
